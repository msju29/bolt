## 什么是Bolt?一个来自爱沙尼亚的"野心家"

Bolt这个名字可能没有Uber那么如雷贯耳,但在欧洲它可是个不折不扣的"狠角色"。

2013年,一个19岁的爱沙尼亚高中生Markus Villig从家里借了5000欧元,在塔林的街头一个一个地招募司机,手工编写了第一版App。当时这个平台还叫Taxify,谁也没想到十几年后它会成长为覆盖50多个国家、600多座城市的出行巨头。

2019年品牌更名为Bolt,正式向全球宣战。现在Bolt的业务已经不仅仅是叫车了:你可以用它点外卖(Bolt Food)、租共享滑板车和电动自行车、租车自驾(Bolt Drive),甚至寄快递。在欧洲、非洲、西亚和拉丁美洲,Bolt已经有超过2亿用户和450万合作伙伴。

2025年2月,Bolt还进军了加拿大多伦多地区,并在华盛顿特区推出了电动滑板车服务。这个成长速度,真的让人不得不刮目相看。

## 为什么选择Bolt?三个让你无法拒绝的理由

### 1. 价格便宜,真的能省钱

在葡萄牙波尔图机场到市区,Bolt大约13.5欧元,Uber要14欧元。别小看这几毛钱,一天下来打三四次车,省下的钱就够买杯咖啡了。

有人说Bolt便宜是因为服务差?我的实际体验完全相反。

Bolt在安全方面下了很大功夫。他们有个专门的"安全团队",500多个真人全天候工作,监控每一笔订单。

App里有紧急求助按钮,一键就能联系到安全团队和当地紧急服务。行程中如果车辆长时间停留,系统会自动检测并发出警报。你还可以把实时位置、车辆信息分享给亲友,让他们随时知道你在哪。

在一些国家,Bolt还推出了"女性专属"功能,让女性乘客可以选择女性司机,增加安全感。
